# 📊 Digital Payment Analysis Dashboard

## 🚀 Project Overview
This project analyzes 5000+ digital payment transactions to identify failure patterns, service performance, and transaction trends.

## 🎯 Objectives
- Analyze transaction success and failure rates
- Identify key failure reasons
- Track monthly transaction trends
- Compare service-wise performance

## 🛠 Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard)

## 📊 Dashboard Features
- KPI Cards (Total Amount, Transactions, Failed, Success)
- Donut Chart (Failure Reasons)
- Bar Chart (Service Analysis)
- Line Chart (Monthly Trend)
- Date Slicer

## 📈 Key Insights
- High failure rates due to server errors and bank declines
- Money Transfer and Bills generate highest revenue
- Seasonal trends observed in transactions

## 📂 Dataset
Synthetic dataset of 5000 transactions
